Visualization¶

Let's compare the countries of Africa based on the number of conflicts that happened from 2007 to 2016. What's the country with the most conflicts?And the least? Do neighbouring countries share the same conflict trend? - This interavtive map will answer the above questions.¶

In [87]:
conf_map = alt.Chart(africa_p, title='Total number of conflicts from 1997 to 2016 by country: Africa').configure_title(fontSize=24).transform_lookup(
    lookup = 'admin',
    from_ = alt.LookupData(data = sum_conf, 
                           key = 'country',
                           fields=['size', 'sum', 'mean'])
).mark_geoshape(color ='lightgray', stroke = 'white', strokeWidth = 1
).encode(
    color = alt.Color('size:Q', title='Number of conflicts:',scale=alt.Scale(scheme='reds', reverse = False)),    
) 

conf_map.encode(tooltip = [
        alt.Tooltip('admin:N', title='Country'),
        alt.Tooltip('size:Q', title='Number of conflicts'),
        alt.Tooltip('sum:Q',title='Number of fatalities'),
        alt.Tooltip('mean:Q', title='Average number of fatalities', format=",.2f")
        ],
               
        opacity = alt.condition(highlight, 
                        alt.value(0.5), alt.value(1)),
        strokeWidth= alt.condition(highlight, 
                           alt.value(3), alt.value(0.5))).add_selection(highlight).properties(width = 800, height = 600).configure_view(strokeWidth = 0)
Out[87]:

Pros: I have used this map over other charts for this task to illustrate the geographical location of the numbers - we can see the geo trends of conflicts. The color palette 'reds' was best suite, as conflicts is smth to be negalivety percepted. Also added Tooltips for more precise info on the number of conflicts and number of fatalities.¶

Cons: While being a good illustration of geographic location of conflicts, the coding the number of conflicts with this scale might be not most illustratiive. There are a few regions with similar colors - so it might be hard to match the slight difference in colors to difference in numbers. Also, even though the Tooltip helps to be more precise, user should memorize the number of conflicts for one country and then hower to a different one to be able to compare two different countries.¶

Overall, for the questions listed above, an interactive map is a good choice.¶

I chose this type of chart for few reasons: it shows the conflict type break-down structure and it is easy-to-read. Initially, I was going to plot conflict break-down only for one country, but the the choice of it would have been hardcoded. Adding a selector for country name lets the user see the frequency of conflict types per any country. I also added alphabetical sort for the dropdown - as it is much easier to find the desired country¶

Pros: It is really clear what type of conflicts were present in a specific country and in which volume(frequency). No extra elements, answers on the stated questions above. User can select any African country from the data available. Can be combined with the first visualization.¶

Cons: User needs to select the country manually. The dropdown could have been configured better (fe placed near the chart title so that it replaces 'selected country'). Also, it is not suitable for comparing 2 countries - only for analyzing the conflict types within the country.¶

Overall, simple chart for a simple question.¶
